Method. Adapted from Anthropic's rightmodel workshop. For each skill we sweep model × thinking × effort (14 cells, matching Anthropic's default), run N trials with an oracle grader, then plot pass rate against cost, latency, and tokens. The winning cell is the lowest credits-per-success with pass rate above a 1/N noise floor.

Costs. Copilot CLI does not bill dollars — it bills tokens converted to credits (post-June 2026) or PRU multipliers (legacy). The simulator uses per-1K-token credit rates and shows both units. Real ledger data would come from gh copilot /usage and the gen_ai.usage.* OpenTelemetry spans emitted by the CLI (reference).
